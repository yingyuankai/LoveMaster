# 四周脱单行动计划

> 基于你的理想型（温婉知性、精致灵气、爱读书运动）和所在城市（上海），量身定制。
> 核心策略：从硬核户外圈转向"轻户外+文艺知性"圈，去对的地方遇到对的人。

---

## 第一周：基础建设 + 初次体验

> 目标：完成个人形象升级，体验 2 个新场景，感受氛围

### 个人形象

- [ ] 换一个清爽利落的发型（可以跟理发师说要"干净有精神"的风格）
- [ ] 整理衣柜，准备 2-3 套适合周末活动的搭配——干净简约风，不用贵但要合身
- [ ] 基础护肤如果还没有就开始：洗面奶 + 水乳 + 防晒
- [ ] 更新朋友圈：清理过期/负能量内容，确保最近可见的朋友圈展示了有趣的你

### 社交媒体准备

- [ ] 如果有社交软件（探探/Soul 等），更新头像和资料。头像用自然光半身照，资料写 2-3 个个性标签
- [ ] 关注小红书/豆瓣上"上海 CityWalk""上海读书会""上海周末活动"等话题
- [ ] 加入 1-2 个感兴趣的活动社群

### 初次体验

- [ ] **CityWalk（周末选一天）**：报名一次法租界/武康路/苏州河沿岸路线。小红书或豆瓣搜"上海 CityWalk 报名"，选人数 10-20 人的小团。目标不是认识女生，而是感受这类活动的氛围和节奏
- [ ] **独立书店（另一天或工作日晚上）**：去一次朵云书院/光的空间/衡山和集/神兽之间，如果有读书会或分享活动更好。先感受一下这个场景

### 本周心态

不急着认识人。这周的任务是"侦察"——看看这些场景什么样、什么人在参加、你自己喜不喜欢。

---

## 第二周：拓展渠道 + 找到你的主场

> 目标：再体验 2 个新场景，从中选出你最喜欢的 1-2 个作为长期主场

### 新场景体验

- [ ] **攀岩/抱石（选一天）**：找一家口碑好的抱石馆做第一次体验
  - 推荐时间：工作日晚上 7-9 点 或 周末下午（人多但不挤，氛围最好）
  - 到了跟前台说第一次来，他们会安排
  - 目标：体验攀岩的感觉，爬几条简单线路，观察馆里的社交氛围
  - 如果旁边有人可以自然聊几句，聊不上也没关系

- [ ] **摄影 Walk 或文艺活动（选一天）**：报名一次摄影 Walk 或花艺/咖啡手作体验课
  - 豆瓣/小红书/大众点评搜索即可
  - 摄影 Walk 结合了你的户外爱好，过渡很自然
  - 手作课小班制互动密度高，容易聊起来

### 复盘 + 选择主场

- [ ] 回顾第一周和第二周体验的 4 个场景（CityWalk、书店、攀岩、摄影/手作）
- [ ] 问自己两个问题：
  1. 哪个活动我自己玩着也开心？（不只是为了认识人）
  2. 哪个活动我遇到的人跟我理想型最接近？
- [ ] 选出 1-2 个作为长期坚持的"主场"

### 稻草人关注

- [ ] 浏览稻草人近期的人文路线短途团（古镇/茶山/寺庙/少数民族文化体验），选一个感兴趣的，为第四周做准备
- [ ] 注意选**人文体验类**路线而非纯户外路线——前者温婉知性女生浓度更高

---

## 第三周：固定节奏 + 开始社交

> 目标：在主场建立存在感，自然认识 3-5 个新朋友（不限性别），加 2-3 个微信

### 主场深耕

- [ ] 重复参加你选定的 1-2 个活动，去同一家馆/同一个组织的活动
  - 攀岩：去同一家馆、同一个时间段，开始混熟面孔
  - CityWalk/摄影Walk：关注同一个组织者的活动，容易遇到重复参加的人
- [ ] 在活动中主动跟人聊天（不只是女生，男生也聊）
  - 目标是成为"这个圈子里友善的人"，而不是"来搭讪的人"
  - 男性朋友同样重要——他们可能会带朋友来，扩大你的社交网络

### 搭话实践

- [ ] 每次活动至少主动跟 2 个陌生人聊天（不限性别）
- [ ] 练习从活动话题过渡到生活话题的能力：
  - 活动相关 → "你经常来吗？"
  - 兴趣延伸 → "你平时还做什么？"
  - 生活层面 → 工作、生活区域、其他爱好
- [ ] 如果聊得来，自然地加微信："聊得挺开心的，加个微信？下次有活动可以一起"

### 朋友圈经营

- [ ] 发 1-2 条有质量的朋友圈：
  - CityWalk 的街拍/建筑照片
  - 攀岩的照片（不用刻意，随手拍就行）
  - 书店/展览的文艺照
- [ ] 风格：展示你在过有趣的生活，不要配大段文字，一句简短的感受就够

### 线上互动

- [ ] 对新加的微信好友，隔一两天自然地聊几句
- [ ] 不要查户口，聊活动延伸话题："上次那条线路我又去试了，终于过了哈哈"
- [ ] 看到对方发的朋友圈，偶尔评论互动（不要每条都评）

---

## 第四周：升级社交 + 制造单独接触

> 目标：从"活动认识"升级到"活动之外也联系"，约 1 个人出来单独喝咖啡

### 稻草人短途团

- [ ] 参加一次稻草人的人文线路周末短途团（1-2 天）
  - 选人文体验类（古镇、茶山、寺庙等），温婉知性型女生浓度高
  - 短途团的好处：高密度相处，一两天下来就从陌生人变成朋友了
  - 在团里自然社交，不用刻意，旅行中人都会更放松、更真实
  - 团结束后加微信很自然

### 主场持续

- [ ] 继续参加你的主场活动，保持固定节奏
- [ ] 这时候你在圈子里已经不算新人了，开始有"熟人"的感觉
- [ ] 如果有认识的人也在，可以一起搭伴参加——社交网络开始产生"滚雪球"效应

### 关键动作：约人出来

- [ ] 在前三周认识的人里，选 1-2 个聊得来的女生，自然地约出来
  - 好的理由："上次你说喜欢那家咖啡店，周末要不要一起去试试？"
  - 或者："有个展览我想去看看，你有没有兴趣？"
  - 轻松的、低压力的一对一——咖啡、展览、逛书店，不要上来就搞烛光晚餐
- [ ] 如果被拒绝也没关系，保持正常的朋友关系继续互动

### 全面复盘

- [ ] 回顾这四周：
  1. 哪个渠道认识的人最多、最投缘？ → 接下来重点投入
  2. 你的搭话能力有没有进步？哪里可以改善？
  3. 有没有遇到心动的人？如果有，下一步怎么推进？
  4. 朋友圈经营得怎么样？有没有女生开始主动找你互动？
- [ ] 根据复盘结果调整下一个月的策略

---

## 长期节奏（第五周起）

四周之后你应该已经找到了适合自己的 1-2 个主场，也积累了一些新朋友。接下来保持这个节奏：

### 每周固定动作

| 频率 | 动作 |
|------|------|
| 每周 1-2 次 | 主场活动（攀岩/CityWalk/读书会/摄影Walk） |
| 每周 1-2 条 | 有质量的朋友圈 |
| 每月 1 次 | 稻草人人文路线短途团或其他新活动 |
| 随缘 | 朋友聚会、饭局（主动跟朋友说"有局喊我"） |

### 关系推进节奏

当你遇到心动的女生时：

1. **第 1-2 周**：在活动中自然互动，加微信
2. **第 2-3 周**：线上保持轻松有趣的聊天
3. **第 3-4 周**：找机会约出来单独见面（咖啡、展览）
4. **之后**：根据她的反馈信号判断是否继续推进（参考恋爱大师 skill 中的"关系推进路线图"）

## 核心原则（贴在墙上）

1. **去对的地方** — 你的理想型在书店、展览、轻户外，不在硬核徒步队伍里
2. **享受过程** — 选你自己也喜欢的活动，脱单是好生活的副产品
3. **先做朋友** — 不要带着"这个人能不能做我女朋友"的滤镜去社交
4. **持续出现** — 社交靠频率和重复，不是一次性搭讪
5. **保持自我** — 你有自己的生活、爱好和追求，这本身就是最大的吸引力
