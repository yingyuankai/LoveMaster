## 二、12 周成长路径模板

### 标准模板（每周 3-5 小时投入）

**第一阶段：基础建设（第 1-3 周）**

- 周目标：完成 [模块名] 的基础内容
- 每周 3-5 个具体行动任务
- 阶段末检查点：[可衡量的成果标准]
- 重点参考：[对应 skill 名称]

**第二阶段：核心突破（第 3-6 周）**

- 周目标：在 [模块名] 上取得明显进步
- 逐步增加挑战难度
- 开始实践场景化练习
- 阶段末检查点：[可衡量的成果标准]
- 重点参考：[对应 skill 名称]

**第三阶段：进阶提升（第 6-9 周）**

- 周目标：拓展到 [模块名] 领域
- 整合前两个阶段的能力
- 更多真实场景实践
- 阶段末检查点：[可衡量的成果标准]
- 重点参考：[对应 skill 名称]

**第四阶段：整合实战（第 9-12 周）**

- 周目标：综合运用所有能力
- 完成 [模块名] 的系统学习
- 制定后续长期成长方向
- 阶段末检查点：[可衡量的成果标准]
- 重点参考：[对应 skill 名称]

### 年龄调整

**18-22 岁调整：**
- 每个阶段可以缩短到 2 周（时间更充裕，试错成本低）
- 任务可以更探索性（"这周去参加一个从没参加过的社团活动"）
- 鼓励多尝试、不怕失败

**23-30 岁调整：**
- 标准节奏，每阶段 3 周
- 任务兼顾工作场景（"这周在一次会议中主动发言一次"）
- 强调效率和可持续性

**30-40 岁调整：**
- 每个阶段可以拉长到 4 周（更注重深度和质量）
- 任务更精准、更高级（"这周练习一次结构化汇报"）
- 减少数量，提升单个任务的质量

**40 岁以上调整：**
- 节奏完全由用户决定，不设硬性时间线
- 任务侧重自我发现和内在成长
- 更多反思类任务

## 三、周行动计划模板

每周计划包含 3-5 个具体任务，格式如下：

```markdown
## 第 N 周行动计划

**本周主题：** [一句话概括]
**所属阶段：** 第 X 阶段 - [模块名]

### 本周任务

1. **[任务名称]**
   - 具体做什么：[详细描述]
   - 为什么做：[和目标的关系]
   - 完成标准：[怎么算做到了]

2. **[任务名称]**
   - 具体做什么：[详细描述]
   - 为什么做：[和目标的关系]
   - 完成标准：[怎么算做到了]

3. **[任务名称]**
   - 具体做什么：[详细描述]
   - 为什么做：[和目标的关系]
   - 完成标准：[怎么算做到了]

### 本周思考题
- [一个和本周主题相关的反思问题]

### 下周预告
- [下周会做什么的简要提示]
```

**任务设计原则：**
- 每个任务最多花 1 小时，大部分 15-30 分钟
- 至少 1 个任务是"在真实场景中做一件事"（不只是看和想）
- 难度递进：第一周最简单，后面逐周增加
- 每个任务都要解释"为什么做"，让用户理解意义
