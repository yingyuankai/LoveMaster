# 四周脱单行动计划（上海·陆家嘴版）

> 你的情况：陆家嘴附近上班，大约 7 点下班
> 你的理想型：温婉知性、精致灵气、爱读书运动
> 核心策略：工作日晚上做轻量活动，周末做重点社交场景

---

## 第一周：基础建设 + 初次体验

### 周一｜形象升级启动

**下班后 7:30-9:00**
- [ ] 去理发店换发型。陆家嘴附近正大广场/国金中心都有不错的理发店
- [ ] 跟理发师说："干净清爽，有精神的风格"，可以提前小红书搜"男生干净发型"存几张参考图

**晚上回家**
- [ ] 整理衣柜，淘汰不合身的旧衣服
- [ ] 在淘宝/优衣库下单 2-3 套基础搭配（干净简约风，参考：白T/浅色衬衫 + 合身长裤 + 白色运动鞋）
- [ ] 如果没有护肤习惯，下单基础三件套：洗面奶 + 水乳 + 防晒

### 周二-周三｜社交媒体准备

**晚上在家 30 分钟**
- [ ] 清理朋友圈（删除负能量/过时内容，确保最近可见的内容展示有趣的你）
- [ ] 小红书关注话题："上海CityWalk""上海读书会""上海周末活动""上海攀岩"
- [ ] 豆瓣关注小组：上海同城活动、上海CityWalk
- [ ] 浏览本周末的 CityWalk 活动，选一个报名

### 周四｜下班后书店体验

**7:00-9:00｜朵云书院·上海中心旗舰店**
- 地址：**浦东新区银城中路501号 上海中心大厦52层**（就在陆家嘴，下班走过去就到）
- 营业时间：10:00-21:30
- 需提前预约：微信公众号"世纪朵云"，每日 7:00-19:00 可约，免费
- 周四白天记得预约！
- 目标：感受氛围，逛一逛，看看这里的人群画像。如果有活动更好
- 下班直达，不用绕路，非常方便

### 周六｜CityWalk 初体验（重点日）

**下午 14:00-17:00｜法租界路线**

推荐路线（小红书/豆瓣报名 10-20 人的组织团）：
- **武康路-安福路-乌鲁木齐中路**：经典文艺路线，独立咖啡店和老洋房密集
- 地铁：2号线陆家嘴 → 换10号线到上海图书馆站，步行即到
- 约 30-40 分钟车程

或者不想过江，也可以选浦东路线：
- **浦江东岸滨江步道**：船厂1862 → 融书房 → 望江驿系列
- 从陆家嘴沿滨江一路走，全程步行可达

**目标：** 不急着认识人，感受 CityWalk 的氛围和参加的人群

### 周日｜休息 + 复盘

- [ ] 回顾：书店和 CityWalk 的体验怎么样？人群跟你的理想型匹配吗？
- [ ] 发一条朋友圈：CityWalk 中拍的好看照片，配一句简短感受
- [ ] 浏览下周的攀岩馆和摄影活动信息

## 第二周：拓展渠道 + 找到主场

### 周二或周三｜下班后攀岩初体验

**7:30-9:30｜岩舞空间攀岩馆（梅奔馆）**
- 地址：**浦东新区世博大道1200号 梅赛德斯-奔驰文化中心B1层**
- 营业时间：周一至周五 13:00-22:00
- 费用：工作日 60元/人（含鞋子租赁）
- 从陆家嘴过去：地铁13号线 → 世博大道站，约 20 分钟
- 不需要预约，直接去跟前台说第一次来
- 目标：体验抱石，爬几条简单线路，观察馆里的社交氛围

**为什么选工作日晚上：**
- 人数适中，不像周末那么挤
- 来的都是下班后的常客，容易遇到固定面孔
- 工作日 60 元比周末 80 元便宜

### 周四｜下班后书店/展览

**7:00-9:00｜选一个去**

选项 A：**浦东美术馆**
- 地址：浦东新区滨江大道2777号（陆家嘴滨江，步行可达）
- 看当前展览信息，提前买票

选项 B：**融书房（浦东图书馆·前滩分馆）**
- 地址：浦东新区东育路500号前滩友城公园内
- 被称为浦东最美图书馆，面朝黄浦江
- 免费，带身份证即可

选项 C：再去一次朵云书院，参加一次他们的读书会/分享活动

### 周六｜摄影 Walk 或手作课（重点日）

**下午 14:00-17:00**

选项 A：**摄影 Walk**
- 豆瓣/小红书搜"上海摄影Walk报名"，选一个法租界或苏州河沿线的
- 结合你的户外爱好，过渡自然
- 参加的女生普遍有审美追求

选项 B：**花艺/咖啡手作课**
- 大众点评搜"上海花艺体验课"或"手冲咖啡体验"
- 选法租界/安福路一带的店，氛围好
- 小班制 6-10 人，互动密度高
- 费用大约 150-300 元

### 周日｜复盘 + 选主场

- [ ] 四个场景都体验过了（CityWalk、书店、攀岩、摄影/手作），回答两个问题：
  1. 哪个活动我自己也玩得开心？
  2. 哪个活动遇到的人跟我理想型最接近？
- [ ] 选出 1-2 个作为长期"主场"
- [ ] 浏览稻草人旅行的人文路线，选一个第四周的短途团报名
- [ ] 发一条朋友圈（攀岩/摄影/手作的照片）

## 第三周：固定节奏 + 开始社交

### 周二或周三｜主场活动·工作日场

**7:30-9:30**

如果主场是攀岩：
- 去岩舞空间，跟上周同一个时间段
- 这次开始注意"熟面孔"——上周也在的人
- 主动跟人聊："上周好像也看到你了？你一般都这个时间来？"
- 跟男生也聊，别只盯着女生

如果主场是书店/读书会：
- 参加朵云书院或其他书店的活动
- 活动结束后是最好的搭话时机："你觉得今天这个分享怎么样？"

### 周四｜下班后轻社交

**7:00-9:00**

- [ ] 去浦东滨江散步（陆家嘴到前滩这一段很舒服）
- [ ] 拍几张好看的照片发朋友圈
- [ ] 或者去融书房看书，习惯在公共空间独处但开放社交的状态

### 周六｜主场活动·周末场（重点日）

**下午 14:00-17:00**

重复参加你选定的主场活动：
- CityWalk：关注同一个组织者的新路线
- 摄影Walk：同一个社群的新活动
- 攀岩：去同一家馆，周末下午场

**这次的社交目标：**
- [ ] 至少主动跟 2 个陌生人聊天（不限性别）
- [ ] 练习话题过渡：活动相关 → 兴趣延伸 → 生活话题
- [ ] 如果聊得来，加微信："聊得挺开心的，加个微信？下次有活动可以一起"
- [ ] 本周目标：加 2-3 个新微信

### 周日｜线上互动 + 朋友圈

- [ ] 对这周新加的好友，发一条轻松的消息（接上活动中的话题）
- [ ] 发一条朋友圈（展示你这周有趣的生活）
- [ ] 复盘：搭话能力有没有进步？哪里可以改善？

## 第四周：升级关系 + 制造一对一

### 周二或周三｜主场活动

**7:30-9:30**
- [ ] 继续你的主场活动
- [ ] 到这周你已经是"常客"了，跟几个人已经混脸熟
- [ ] 如果之前加了微信的人也在，主动打招呼一起玩

### 周四｜约人出来

**7:00-9:00**

- [ ] 在前三周认识的人里，选一个聊得来的女生，约出来
- [ ] 好的邀约方式：
  - "我发现陆家嘴滨江新开了一家咖啡店，看起来不错，周末要不要一起去试试？"
  - "浦东美术馆最近有个新展，你有兴趣吗？"
  - "你上次说想去的那个地方，要不这周末一起？"
- [ ] 如果这周约不出来也别急，保持互动，下周再试

### 周六-周日｜稻草人人文短途团（重点）

**整个周末**

- [ ] 参加你第二周报名的稻草人人文路线短途团
- [ ] 推荐线路类型：
  - 江南古镇（西塘/南浔/同里）
  - 莫干山/安吉人文体验
  - 茶山/寺庙文化体验
- [ ] 在团里的社交策略：
  - 旅行中人都更放松，主动和大家聊天很自然
  - 帮忙拍照是绝佳的搭话方式
  - 吃饭时主动坐在不认识的人旁边
  - 团结束后加感兴趣的人微信

### 周日晚上｜全面复盘

- [ ] 回顾这四周：
  1. 哪个渠道认识的人最多、最投缘？ → 下个月重点投入
  2. 搭话能力进步了多少？还紧不紧张？
  3. 加了多少新好友？有没有心动的人？
  4. 朋友圈有没有女生开始主动互动？
- [ ] 制定下个月的计划（延续有效的渠道，调整无效的）

## 核心原则（贴在手机备忘录里）

1. **去对的地方** — 你的理想型在书店、展览、轻户外
2. **享受过程** — 选你自己也喜欢的活动
3. **先做朋友** — 别带"能不能做女朋友"的滤镜
4. **持续出现** — 社交靠频率和重复
5. **保持自我** — 有自己的生活就是最大的吸引力
