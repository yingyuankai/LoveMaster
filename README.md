# LevelUp

**男生版自我提升操作系统**

LevelUp 是一个基于 Claude Code 插件的 AI 个人成长教练，专为男生设计。你只需要告诉它你的目标，它就会诊断你的现状、生成个性化的成长路径，并在每周 check-in 中持续陪你进步。不说空话，只给行动方案。

---

## What is LevelUp | 什么是 LevelUp

LevelUp 是一个**目标驱动**的个人成长系统。它不是一本静态的"男生必读手册"，而是一个能理解你、跟你对话、为你量身定制方案的 AI 教练。

**工作流程：**

1. 你设定一个目标（比如"我想脱单"、"我想变得更自信"）
2. 系统通过对话诊断你的现状（年龄、城市、性格、社交习惯等）
3. 生成个性化的成长路径和每周行动计划
4. 持续提供指导，每周 check-in 跟踪进展

**覆盖领域：**

- 恋爱社交 — 从认识到经营，全流程指导
- 男性魅力 — 气场、眼神、肢体语言、领袖气质
- 形象管理 — 穿搭、发型、护肤、体态
- 情绪管理 — 焦虑、自信、内耗、安全感
- 沟通表达 — 聊天、演讲、讲故事、职场沟通

LevelUp 以 Claude Code 插件的形式运行，安装后即可在 Claude Code 中直接使用。

---

## Core Features | 核心特色

- **目标驱动** — 不是"你该学什么"，而是"你想成为什么"。一切从你的目标出发。
- **个性化** — 根据你的年龄、城市、性格、现状定制建议，没有千篇一律的模板。
- **可执行** — 每个建议都具体到行动。不说"你要更自信"，而是告诉你今天做什么、怎么做。
- **持续陪伴** — 每周 check-in，长期跟踪你的成长轨迹，及时调整方案。

## Modules | 模块一览

| 模块 | 名称 | 说明 |
|------|------|------|
| `onboarding` | 入门诊断 | 首次使用时的诊断对话，了解你的现状和目标，生成个性化成长路径 |
| `love-master` | 恋爱大师 | 全流程恋爱指导：认识女生、聊天技巧、约会策划、关系经营、矛盾处理 |
| `alpacino-charm` | 魅力修炼 | 以阿尔·帕西诺为灵感的男性魅力训练：气场、眼神、声音、肢体语言 |
| `image-craft` | 形象管理 | 穿搭指南、发型建议、护肤方案、体态改善，打造你的个人形象 |
| `inner-game` | 内心建设 | 焦虑管理、自信培养、依恋理论、情绪调节，解决内耗和自我怀疑 |
| `social-voice` | 沟通表达 | 聊天技巧、讲故事能力、职场沟通、公众表达，告别"不会说话" |

## Quick Start | 快速开始

### 1. 安装插件

```bash
claude plugin add /path/to/LevelUp
```

将 `/path/to/LevelUp` 替换为本项目在你电脑上的实际路径。

### 2. 启动 LevelUp

在 Claude Code 中运行：

```
/levelup
```

### 3. 完成诊断

系统会像朋友聊天一样问你几个问题，了解你的现状。整个过程大约 3-5 分钟。

### 4. 获取成长路径

诊断完成后，你会收到一份个性化的成长路径和第一周的行动计划。

### 5. 开始行动

按照计划执行，有问题随时问。每周记得 check-in。

## Commands | 命令

| 命令 | 说明 |
|------|------|
| `/levelup` 或 `/levelup start` | 开始诊断或查看当前状态 |
| `/levelup checkin` | 每周回顾，汇报进展，获取下周计划 |
| `/levelup plan` | 查看你的成长路径和当前进度 |
| `/levelup reset` | 重新设定目标，重新开始诊断 |

## Examples | 示例对话

**示例 1：首次使用**

> **你：** /levelup
>
> **LevelUp：** 兄弟，欢迎来到 LevelUp！接下来我会问你几个问题，帮你摸清现在的状态，然后给你量身定一个成长计划。先聊聊，你现在最想改变的是什么？
>
> **你：** 我想脱单，但是不知道怎么认识女生
>
> **LevelUp：** 收到。你平时的社交圈大概是什么样的？工作之外一般怎么认识新朋友？

**示例 2：聊天求助**

> **你：** 她回了个"哈哈"，我不知道怎么接了
>
> **LevelUp：** "哈哈"不代表没兴趣，但确实说明当前话题能量不够。这时候不要追问"怎么了"，换一个她可能有表达欲的话题。比如你们之前聊过什么她特别感兴趣的？我帮你组织一下回复。

**示例 3：每周 check-in**

> **你：** /levelup checkin
>
> **LevelUp：** 这周过得怎么样？上周给你的三个任务完成情况如何？1）每天跟一个陌生人打招呼 2）整理衣柜，淘汰不合身的衣服 3）睡前做 5 分钟呼吸练习。一个一个说，咱们复盘一下。
